# docker-golang-basic-image

## Objetivos
O objetivo deste projeto é que ao subirmos o container referente a essa imagem, que seja gerada uma saída de tela com a frase "Full Cycle Rocks!!". Além disso, o tamanho máximo da imagem deve ser de 2MB

## Sobre o projeto
Este projeto é referente ao primeiro desafio do módulo sobre Docker do curso Full Cycle 3.0.
Basicamente o projeto se resume a criação de uma imagem Docker baseada na linguagem Go.

## Endereço da imagem no 
https://hub.docker.com/repository/docker/igoorj/fullcycle/general

## Subindo um container dessa imagem
docker run igoorj/fullcycle
